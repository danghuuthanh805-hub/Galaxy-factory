# 🌌 Galaxy Factory

Welcome to **Galaxy Factory**, an exciting space-themed idle/clicker game where you build and upgrade your cosmic empire from scratch! 

This game is fully developed in a single, lightweight `index.html` file using vanilla web technologies.

## 🎮 Game Features
* **100% HTML5 Canvas Rendered:** The entire game graphics, animations, and visual effects are drawn programmatically using JavaScript Canvas API. **No external images or assets are required!**
* **Classic Idle/Clicker Mechanics:** Click to generate space energy/resources and expand your galaxy.
* **Progressive Upgrades:** Spend your cosmic energy on automated space factories to boost your Clicks Per Second (CPS).
* **Auto-Save System:** Powered by `localStorage`, your galactic progress is saved automatically so you never lose your hard-earned upgrades.

---

## 🛠️ Built With
* **HTML5 Canvas:** For rendering smooth, lightweight, and fast vector game graphics.
* **CSS3:** For setting up the immersive dark space background and layout wrapper.
* **JavaScript (ES6):** For managing the game loop, upgrade logic, mathematical formulas, and state persistence.

---

## 🚀 How to Play
1. **Click** the core generator on the canvas to start producing space resources.
2. **Buy Upgrades** from the panel as soon as you have enough resources to unlock automated space factories.
3. **Idle & Multiply:** Watch your cosmic resource count grow exponentially over time!
4. **Refresh Safely:** Don't worry about closing the tab; your factory data is safely preserved in your browser.
