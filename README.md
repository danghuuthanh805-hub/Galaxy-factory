# 🌌 Galaxy Factory

Welcome to **Galaxy Factory**, an exciting space-themed idle/clicker game where you build and upgrade your cosmic empire from scratch! 

This game is fully developed in a single, lightweight `index.html` file using **HTML, CSS, and JavaScript**.

---

## 🎮 Game Features
* **Classic Idle/Clicker Mechanics:** Click to generate space energy/resources and expand your galaxy.
* **Progressive Upgrades:** Spend your earnings on automated space factories to boost your Clicks Per Second (CPS).
* **Responsive Cosmic UI:** A clean and immersive dark space theme tailored for modern browsers.
* **Auto-Save System:** Powered by `localStorage`, your galactic progress is saved automatically so you never lose your hard-earned upgrades.

---

## 🛠️ Built With
* **HTML5:** For the structural layout of the control panels and upgrade buttons.
* **CSS3:** For the beautiful stellar visuals and animations.
* **JavaScript (ES6):** For the complex game logic, progress formulas, and state management.

---

## 🚀 How to Play
1. **Click** the core generator to start producing space resources.
2. **Buy Upgrades** as soon as you have enough resources to unlock automated space factories.
3. **Idle & Multiply:** Watch your cosmic resource count grow exponentially over time!
4. **Refresh Safely:** Don't worry about closing the tab; your factory data is safely preserved.
